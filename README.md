# 🚀 FairHire AI

AI-powered bias-free hiring system with fairness scoring and explainable decisions.

---

## 🌐 Live Demo
https://fairhiring-hub.lovable.app

---

## 🧠 Problem
Hiring systems often introduce unconscious bias based on gender, age, or background.

---

## 💡 Solution
FairHire AI evaluates candidates purely on skills and experience, while detecting and reporting potential bias signals.

---

## ✨ Features
- Resume parsing using AI
- Candidate scoring system
- Bias detection (gender, age, location)
- Fairness score calculation
- Explainable hiring decisions
- Supabase database integration

---

## ⚙️ Tech Stack
- n8n (workflow automation)
- Google Gemini API
- Supabase (database)
- Lovable (frontend)

---

## 🔄 Workflow
1. Receive candidate data
2. Extract resume text
3. Clean and preprocess data
4. AI evaluation (Gemini)
5. Parse structured output
6. Detect bias signals
7. Calculate fairness score
8. Store in database
9. Return results

---

## 📊 Output Example
- Score: 75
- Decision: SHORTLISTED
- Fairness Score: 90
- Bias Status: UNBIASED
